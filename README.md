# Marcelo-Challengue

## Install project

    npm i

## Build project

    ng build

## Usage

    ng serve
    In a browser open http://localhost:4200/

## Built With:

    Angular CLI version 9.1.9.
    Angular Material version 9.2.4.

## Testing

    # Services Unit Tests
        ng test

    # End to End Tests
        ng e2e
